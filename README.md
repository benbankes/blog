### Setup

To setup a development environment with ansible, run the ansible playbook.

1. Get ansible if you do not already have it

    ```
    sudo apt update
    sudo apt install ansible
    ```

1. Run the playbook

    ```
    cd ansible
    ansible-playbook setup-development-environment.yml
    ```
